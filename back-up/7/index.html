이메일 인증 유의사항

  • parking fairy 가입인증 메일은 가입 요청 후 코드 입력 시간(10분) 내로 발송됩니다.
  • “인증 메일이 오지 않는 경우” : 코드 입력 시간(10분)이 지나도록 가입 인증 메일을 받지 못했을 경우 스팸메일함을 확인 부탁드립니다.
    이후에도 지속적으로 오류 발생 시 본인의 이메일을 통해 xxx123@mju.ac.kr 으로 아래 정보를 전달해주시기 바랍니다.
    1. 이메일 제목: “인증 메일이 오지 않습니다.”
    2. 이메일 본문: 고객 이름/삼성계정 ID/학년정보/인증메일 신청 화면 스크린샷 캡쳐
    (ID는 이메일 또는 전화번호 형식으로 구성되어 있습니다. ID, 메일주소 등 상기 정보 오기입 시 가입 절차가 지연될 수 있으니 반드시 정확한 내용으로 기입 부탁 드립니다)
  • “이미 인증받은 이메일로 이메일 인증 불가”라고 뜨는 경우 : 보통 두 개 이상의 계정을 가지고 있어 기존에 인증 받았던 계정 외 다른 계정으로 로그인을 시도할 시 다시 이메일 인증을 받으라는 안내가 뜨게 됩니다.
    1. 계정 찾기를 통해 계정 확인 후 로그인 시도 바랍니다.
    2. 이후에도 오류 발생 시 성함과 휴대폰 번호, 계정 ID 및 인증 시도하신 이메일을 xxx123@mju.ac.kr으로 보내주시기 바랍니다.
    (로그인 시 크롬 사용을 권장 드리며 쿠키 및 캐쉬 삭제 후 로그인 부탁드립니다)
